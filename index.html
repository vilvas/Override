Override 1.0 Documentation

Installation

Override is easy to install into your WordPress plugin, just like any other plugins. We can install Override in two formats, 1. Upload within WordPress or 2. Upload with FTP/SFTP Software.

Upload within WordPress

  1. Login to WordPress Admin
  2. Hover over Plugins menu
  3. Click on Add New
  4. Then click on Upload Plugin on top
  5. Click Choose File
  6. Locate override.zip on your computer and upload

Upload with FTP/SFTP Software.

  1. Choose your prefer FTP/SFTP software. (We Love FileZilla!)
  2. Login to your Host
  3. Locate your Plugins Folder within your WordPress directory
  4. wp-content > Plugins
  5. Upload Override folder in there. (Make sure is unzipped)
  6. Login to WordPress Admin
  7. Click Plugins
  8. Activate Override

General

Setup any general settings available, that will fit your need.

Logo

  1. Hover over Settings
  2. Click on Override
  3. Click on General tab. It's active by default
  4. Upload your Logo by clicking on the Upload button on "Logo" Section of the tab.

Customization

Override allows you to customize many aspects of your login/registration page, easy to upload a background image and Unlimited Coloring. You will see all these settings in Customization Tab.

Background

  1. Upload Background Image Optional
  2. Upload your Background Image by clicking on the Upload button on "Background Image" Section of the tab.
  3. Select Background Color Optional
  4. Select Box Background Color Optional
  5. This is the surrounding of the Login / Registration sectio

Fonts

  1. Select Label Color This is the Username, Password etc.
  2. Select Link Color

Buttons

  1. Select Button Color
  2. Select On Hover Color CSS3 Transition already Included 0.4S

Custom CSS/JS

You can also overwite or enter a new CSS style and Javascript in Custom CSS/JS tab. Enter your code same way you would do in .css file or .js file.
